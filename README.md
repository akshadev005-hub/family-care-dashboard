# 🏠 Family Care Dashboard

A smart family reminder system built using Flask and SQLite.

## 💡 Features
- Add medicine reminders for Dad & Mom
- Separate dashboard sections
- Live reminder tracking
- Modern UI with gradient design
- SQLite database storage

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite

## 🚀 Run Locally
```bash
pip install flask
python app.py
